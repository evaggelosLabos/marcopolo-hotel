🏨 Marcopolo Hotel — Hotel Website
A modern, production-ready hotel website built with Next.js and TypeScript, designed for a real hospitality client in Greece.
Live Site
👉 https://marcopolocorfu.com
Features

Full hotel website with rooms, amenities, and gallery
Booking enquiry system with automated email confirmations
SEO-optimised pages with structured metadata for Google search visibility
Fully responsive design for mobile, tablet, and desktop
Optimised images and performance-first architecture

Tech Stack

Framework: Next.js 14 (App Router)
Language: TypeScript
Styling: CSS / Tailwind
Deployment: Vercel / Azure

Getting Started
bashnpm install
npm run dev
Open http://localhost:3000 to view it in your browser.
